Common questions
Which platforms does HOOK support? ▾

HOOK supports X (Twitter), TikTok, Instagram, and LinkedIn. All four platforms are available on every plan. You can connect multiple accounts per platform — 1 on Free, 3 per platform on Creator, and unlimited on Pro.

Can I really connect multiple accounts per platform? ▾

Yes — within each platform's official API terms. Each account connects via its own OAuth flow. Creator tier allows 3 accounts per platform (e.g. 3 X accounts + 3 Instagram accounts). Pro tier is unlimited. Multiple accounts can be assigned to the same or different campaigns.

Can I cancel anytime? ▾

Yes. Cancel at any time from your account settings — no questions asked. You'll retain access until the end of your billing period. Annual plans are non-refundable after 14 days but can be cancelled to prevent renewal.

Does HOOK use official platform APIs? ▾

Yes. HOOK uses only official APIs: X API v2, TikTok Content Posting API, Instagram Graph API, and LinkedIn API v2. No browser automation, no scraping. Your accounts are safe and compliant.

Is HOOK GDPR compliant? ▾

Yes. HOOK is built for the UK/EU market. All OAuth tokens are encrypted at rest (AES-256-GCM). You can request a full data export or account deletion at any time from Settings. Data is deleted within 30 days of a deletion request. Full privacy policy available.
